### **Miller-Rabin Primality Test**  

**Описание:**  
Этот код реализует **вероятностный тест Миллера-Рабина** для проверки чисел на простоту.  

**Основные функции:**  
- Проверяет, является ли число простым или составным.  
- Поддерживает **быстрое возведение в степень по модулю** (оптимизация для больших чисел).  
- Позволяет задавать количество раундов теста (`k`) вручную или автоматически (`k = log(n)`).  

**Использование:**  
1. **Ручной ввод:**  
   - Введите числа для проверки.  
   - Выберите количество раундов (`k`).  
2. **Чтение из файла:**  
   - Укажите файл с числами (каждое число на новой строке, можно добавить `k`).  
3. **Результаты:**  
   - Вывод в консоль и сохранение в файл **`Result.txt`**.  

**Пример вывода:**  
```
17 5 - prime number  
25 5 - composite number  
```  

**Применение:**  
- Криптография (RSA, генерация ключей).  
- Тестирование алгоритмов, работающих с простыми числами.  

**Запуск:**  
```bash
python3 miller_rabin.py
```
